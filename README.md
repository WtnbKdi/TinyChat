# TinyChat
LAN内で使用することができるチャットツールです。

各操作は、コマンド.txtのコマンドをクライアント、サーバー間で送受信することで実現しています。
# 機能
- 文字列送受信
- 部屋の作成
- ユーザー名の重複チェック
